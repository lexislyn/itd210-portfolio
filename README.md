Milestone 5 — Contact Page (contact.html)
I have included a webform that is connected to Formspree. When you submit the form, a success state appears thanking the user for filling out the form. There's also a social media section, where the icons scale larger when you hover over them. 

Final Reflection
What is the strongest part of your final site and why?
  I believe both the key issues page and the events page are the strongest parts of my site. For the key issues page, I'm really proud of how I learned to use Tailwind CSS and that the page is mostly Tailwind CSS. Plus, I'm excited that the filtering function works on that page. It adds a really fun interactive element to that page. I also love how the events page turned out. I love that I was able to add a modal function that mimics an event sign up form and makes the calendar an interactive element as well, not just a static calendar. 

What did you learn about responsive design and interactivity this semester?
  I learned so much about transitions and animation this semester, that now each page has those elements in there which allows for a much more interactive experience than I was capable of making last year. My responsive design skills have also improved immensely and I'm proud of how my website looks on both a laptop and a phone. 

Which piece of feedback changed your approach the most?
  I now know the difference between serif and sans serif font, so I will take that information into all of my future websites and make sure that I use sans serif font for body text in order to make my site more clear and readable for users. And although maybe not direct feedback, I found the advice of adding 2-3 interactive elements per page helpful as a guidemark. It turned my focus onto making my website more interactive on every page.

If you had another month, what would you add or refine next?
  If I had another month, I would want to add a feature to my calendar page, that after you register for an event with the modal, you can get a calendar download link that you could add to a google or apple calendar on your phone or laptop. I think that would be a cool, useful feature that could make my page stand out more. I would also focus on ways to make my webpage load faster. I feel like the amount of images I'm using causes the page to load really slowly, so I would work on improving that. I would also improve the dark mode feature, because while I'm happy with what I have now, it's definitely my first time adding a dark mode feature and I'm sure it's not perfect. I would also add a search feature, since I feel like most websites have that. It would just add credibility to my site.
